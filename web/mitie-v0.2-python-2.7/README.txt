This is the binary distribution of MITIE v0.2

The project is hosted at https://github.com/mit-nlp/MITIE.  See that page for
the full documentation and source code.


This package contains shared libraries compiled for 64bit Linux and also
Windows (32bit and 64bit) as well as three Python 2.7 example programs.  You
should be able to run the python examples by simply executing them on the
command line.  For example, type:
    python ner.py

None of the examples take any command line arguments.
